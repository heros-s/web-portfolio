# Pipeline de Dados Automatizada para Campanhas de Mailing

Categoria: Engenharia de Dados · Automação
Descrição curta (hero): Pipeline automatizada que eliminou a execução manual de mailings e aumentou a eficiência das campanhas.
Impacto: Redução do tempo operacional por ciclo e aumento de até 27% na eficiência das campanhas.
Mídias: Código, prints do processo, logs de execução (mock)
Ordem: 4
Problema: Processo manual recorrente, suscetível a erros, atrasos e indisponibilidade operacional.
Slug: pipeline-dados-mailing
Solução: Script que executa queries, gera CSVs e envia automaticamente para o discador via SFTP em ciclos horários.
Status: Descontinuado*
Tecnologias: Python, PostgreSQL, Pandas, Paramiko, SFTP
