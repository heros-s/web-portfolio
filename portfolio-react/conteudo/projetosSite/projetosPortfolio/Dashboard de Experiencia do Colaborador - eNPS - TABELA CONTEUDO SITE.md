# Dashboard de Experiência do Colaborador (eNPS)

Categoria: BI · People Analytics
Descrição curta (hero): Dashboard interativo que substituiu apresentações manuais extensas e viabilizou análises contínuas de clima organizacional.
Impacto: Economia de tempo do RH e maior frequência de uso dos dados para decisões estratégicas.
Mídias: Prints do dashboard, vídeo de navegação (mock)
Ordem: 5
Problema: Análises feitas manualmente a cada pesquisa, com alto esforço e pouca reaproveitabilidade.
Slug: dashboard-enps
Solução: Painel dinâmico com métricas de favorabilidade, evolução temporal, liderança e deep dive de comentários.
Status: Em uso
Tecnologias: Power BI
